# mapboxgljs-starter

![README02](./img/README02.png)

![README05](./img/README05.png)

Start MapboxGLJS easily.  
- [MapboxGLJS v0.50.0](https://www.mapbox.com/mapbox-gl-js/api/) 
- [webpack v4.20.2](https://webpack.js.org)  
- node v10.0.0
- npm v6.4.1

<br>

## Usage

![README04](./img/README04.png)

<br>

Install package

```

npm install

```

<br>

build

```

npm run build

```

<br>

dev

```

npm run dev

```
