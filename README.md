# mapboxgljs-starter

![README02](img/README02.png)

![README05](img/README05.png)

Start MapboxGLJS easily.  
- [MapboxGLJS v1.5.0](https://www.mapbox.com/mapbox-gl-js/api) 
- [webpack v4.39.1](https://webpack.js.org)  
- node v13.1.0
- npm v6.13.0

<br>

## Usage

![README04](img/README04.png)

<br>

Install package

```

npm install

```

<br>

build

```

npm run build

```

<br>

dev

```

npm run dev

```
